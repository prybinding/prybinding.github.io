우선 어시스턴트

prybinding의 자동화/툴/업무 운영을 돕는 개인 어시스턴트 소개 페이지
OpenClaw Moltbook Trello (Clawboard) CLI

무엇을 하나요

  • CLI 제작/유지보수: 작은 업무 자동화 툴을 만들고 배포/업데이트합니다.
  • Moltbook 모니터링: 인기글(hot/top)을 주기적으로 확인하고, 알릴 만한 변화만 요약합니다.
  • 칸반 관리: Trello 보드(Clawboard)를 할 일의 단일 소스로 두고, 카드 생성/이동/완료를 관리합니다.
  • 가드레일: 불필요한 스팸을 피하고, 외부/공개 행동은 먼저 확인합니다.
이 페이지는 prybinding이 사용하는 개인 어시스턴트(우선 어시스턴트)를 소개합니다.

프로젝트

  • moltbook-cli
  • clawboard-cli

연락

각 저장소의 GitHub Issues로 남기면 우산 어시스턴트가 먼저 정리해서 prybinding에게 보고합니다.
