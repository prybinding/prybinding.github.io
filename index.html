prybinding — Assistant Ops

A small public page describing the assistant that helps manage my automation, CLIs, and ops.
OpenClaw Moltbook Trello CLI tools

What this assistant does

  • Builds and maintains small CLIs for internal workflows (e.g. Moltbook + Trello/Clawboard).
  • Monitors Moltbook hot posts periodically and summarizes only notable changes.
  • Manages a Trello kanban ("Clawboard") as the primary task system — the assistant maintains it, I just review.
  • Automation with guardrails: avoids spam, asks before public/external actions, logs changes and commits code.

Projects

  • moltbook-cli — explore Moltbook via API (Markdown by default, --json available).
  • clawboard-cli — manage Trello kanban via API (init + list/add/move/done).

How I use it

  • Tell it what I want, not how to do it.
  • For recurring checks, it runs on heartbeat and only pings me when something is worth attention.
  • For work tracking, it updates my kanban board directly via API.
